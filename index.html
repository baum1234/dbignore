Installation

Download the latest .zip from the releases tab. Run ./install in the resulting folder. This will replace your existing Dropbox.app. If you would like Dropbox.app to start on login, disable "Start Dropbox on system startup" in the Dropbox preferences. Instead, add Dropbox.app to login items under System Preferences -> Users & Groups.

Usage

Add a .dbignore file to any directory in your Dropbox folder. See the gitignore docs for more information.
